
INSTRUCCIONES PARA USAR LAS IMÁGENES OPTIMIZADAS
================================================

1. Extrae este ZIP en tu computadora
2. Ve a tu proyecto: assets/images/products/
3. HAGO BACKUP de las imágenes originales (cópialas a otra carpeta)
4. Reemplaza todas las imágenes con las de esta carpeta
5. Sube los cambios a Netlify

IMPORTANTE:
- Las imágenes están optimizadas para carga rápida
- Mantienen buena calidad visual
- Pesan mucho menos que las originales
- Si usaste formato WebP, actualiza las extensiones en el código
